# 개요
(주)태성환경연구소에서 2개월간 전산팀 현장실습을 진행하면서 만든 프로그램입니다. 센서평가팀에서 사용할 수 있는 센서데이터 시각화 웹 프로그램 개발을 요청받았습니다. 그리고 첫 프로젝트를 완료한 이후에 두번째 프로젝트를 간단하게 진행하고 현장실습이 종료되었습니다.
회사에서 만들었던 프로그램이기 때문에 전체 코드를 공유할 수 없습니다.

## 프로젝트 소개 - 센서데이터 시각화 웹 프로그램
센서평가팀에서 기존에 사용하던 프로그램은 python으로 간단하게 제작된 프로그램으로 실시간으로 업데이트를 할 때 마다 페이지 전체가 다시 로딩되는 형식으로 구성되어 있었습니다. 따라서 사용자가 확인할 때 불편한 부분이 있다는 요청을 알게 되었고, 이 문제를 해결하면서 다른 기능들도 추가적으로 개발했습니다.

### 1. 실시간 데이터 확인 페이지
<img width="1891" height="840" alt="Image" src="https://github.com/user-attachments/assets/af945753-f5b6-45b2-9d0d-8d886e7defa4" />
실시간으로 데이터를 확인할 수 있는 페이지입니다.

- 기능
  - 각 센서 별도로 활성화 기능 (사이드바에서 클릭 시 차트가 하나씩 표시)
  - 최근 몇분까지 저장된 데이터를 가지고 올지 설정 (각 차트별로 설정이 가능)
  - 활성화된 센서에 대해서 최근 몇분까지 보여줄지 설정
  - 실시간으로 차트갱신 (일정 초단위로 갱신)
  - 보고싶은 내용만 확인

- 고민사항

<img width="1832" height="328" alt="Image" src="https://github.com/user-attachments/assets/588587e0-9621-4256-90e3-20ba94ced7c4" />

  - 회사DB의 내용을 임의로 수정할 수 없었기 때문에 데이터가 갱신되는 주기는 확인할 수 없었고, 실시간 데이터 페이지는 갱신되는 주기에 맞춰서 차트에 보여져야 하기 때문에 고민을 했습니다.
    - WebSocket을 사용해서 서버와 프론트 양방향 통신을 연결하고 서버측에서 Polling방식을 이용해서 일정 주기마다 DB에 접근해서 갱신된 데이터의 내용을 프론트로 전송해줬습니다.
